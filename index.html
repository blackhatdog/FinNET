Installation & Setup


Your system will require python installed,along with networking capabilities to run the software.

#1:Requirements
You will require python flask,to install it,run :

pip install Flask

For database management, you will require TinyDB.To install,run:

pip install tinydb


#2:Running the script
Now,navigate to the source folder and run 'main.py' by using 'python main.py' from the shell or terminal of your device.
